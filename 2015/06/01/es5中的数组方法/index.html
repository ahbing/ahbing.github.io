es5中的数组方法

发表于 2015-06-01   |   分类于 js   |  

写代码采用模块化，当然写博客也应该采用模块化。我把js数组的整理分成两篇文章，可以控制没一篇的长度，也方便日后查阅。这篇就整理es5中的一些数组方法。

forEach

forEach用来循环数组，回调函数接受三个参数，分别是数组元素，数组索引，数组本身

使用forEach求和

数组每个元素都增加1

map

map 方法调用操作数组的函数之后，返回一个数组。

map里调用的函数也是接受三个和forEach的参数：数组元素，数组索引，数组本身；

运行结果

filter

filter过滤器。
在filter调用的函数里设置过滤条件，并返回符合条件的元素的数组

返回小于等于3的数组元素

返回索引为非奇数的元素

every 和 some

every 和 some 返回一个布尔值，用来检测数组是否符合某种条件。

every遍历数组过程中遇到false 就直接返回false，结束遍历。(一假即假)；
some则是遇到第一个true，就停止遍历，返回true; (一真即真)

indexOf 和 lastIndexOf

indexOf 和 lastIndexOf 搜索整个数组中给定的值，返回找到的第一个值的索引，
不同的是 indexOf从头开始，而lastindexOf则是从尾开始。如果没有找到返回-1

indexOf 和 lastIndexOf 都接受第二个参数，第二个参数表示开始搜索的位置，当然也可以是负数，负数表示相对数组末位的偏移量，一样的-1表示最后一个元素。

字符串也有相应的indexOf 和 lastIndexOf 方法。

使用indexOf的第二个参数，在数组里查找所有出现v，并返回一个包含匹配索引的函数
