ejs用于在模板填充数据，有三种情况

  • <% code %>：JavaScript 代码。比如forEach循环输出
  • <%= code %>：不解释html的标签，比如<h1>,会原样输出
  • <%- code %>：会解释html标签<h1>hello<h1> ,输出标题号的hello
